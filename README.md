# Is_Devamsizligi_Tahmini
Bu projede UCI Makine Öğrenmesi Veri Havuzu'nda (UCI Machine Learning Repository) yer alan 'İş Yerinde Devamsızlık' (Absenteeism at work) veri seti kullanılarak iş yerindeki devamsızlık durumunlarını öngörmeyi hedeflemek için, veri bilimi ve makine öğrenme yöntemleri kullanılmıştır.
